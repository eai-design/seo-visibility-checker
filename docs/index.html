Batch SEO Visibility Checker

批量檢查網站是否公開可抓取

前端用於整理 URL、查看 CSV 報告；真正抓取由 Python 腳本或 GitHub Actions 執行，避免瀏覽器 CORS 限制。

Google URL Inspection

1. 貼上要檢查的 URL

python3 batch_seo_visibility_checker.py sites.txt docs/reports/latest.csv

2. 查看報告

可上傳 CSV，也可讀取 GitHub Actions 產生的 docs/reports/latest.csv。

建議驗證流程

  1. 先用此工具批量初篩，找出 flags 不是 ok 的 URL。
  2. 每個網站抽樣首頁、分類頁、文章/產品頁，在 Google Search Console 做 URL Inspection → Test Live URL。
  3. 對異常站點檢查 robots.txt、meta robots、CDN/WAF、防火牆、JS 渲染、登入限制。

官方依據：Google crawlers overview、Verify Googlebot。
